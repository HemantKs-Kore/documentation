Your Quick Start Guide
Start by Adding Data
SearchAssist allows you to add different types of data such as FAQs, Actions, Content, Structured Data from various sources.
Index Data
SearchAssist allows you to Index the data that you want to make searchable.
Optimize Search Results
SearchAssist allows to optimize the search results through Synonyms, Weights, Stop Words, Rules, Facets.
Design Search Experience
SearchAssist allows you to customize the search experience and design the search interface based on the business context.
Test the Application
SearchAssist allows you to validate the configuration and search experience before deploying the app by just clicking on the Test button.
Fine-Tune Relevance
SearchAssist Allows you to fine-tune search results further with the result ranking actions such as pinning, boosting, lowering and hiding.
